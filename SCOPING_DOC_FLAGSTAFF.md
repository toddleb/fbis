## 10. WHAT WE'LL BUILD FOR YOUR DEMO

Based on your answers, we'll customize a working demo of FBIS tailored to your operation. Here's what the platform already does today:

### Current Capabilities (Working Now)

| Feature | Description |
|---|---|
| **Real-time inventory dashboard** | See all inventory by location, category, and risk level at a glance |
| **Expiry risk monitoring** | Lots color-coded by urgency — red (≤3 days), amber (4-7 days), green (safe) |
| **AI action recommendations** | System recommends specific redistributions with "meals saved" impact scores |
| **Lot-based tracking** | Every donation tracked as a lot with source, receipt date, expiration, location, condition |
| **Intake form** | Log new donations with category, quantity, expiration, source, storage location |
| **Actions execution** | One-click execution of AI recommendations with running meals-saved counter |
| **Distribution station** | Tablet-optimized split-panel interface — browse inventory, build distribution ticket, complete distribution |
| **Nutrition balance indicator** | Visual indicator showing which food groups are covered in each distribution |
| **Coordinator console** | Warehouse-level overview — all stations, locations, alerts, pending actions |
| **FEFO sorting** | All inventory views automatically sorted First Expired, First Out |
